# House Price Prediction Model

## Overview

This project applies machine learning regression techniques to predict housing prices using property characteristics and housing market data.

The objective was to compare multiple regression models and identify the most effective approach for accurate house price prediction.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

## Dataset

The project uses housing market data containing information such as:

- Area
- Number of Bedrooms
- Number of Bathrooms
- Stories
- Parking Spaces
- Furnishing Status
- Additional Property Features

## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Data Preprocessing
5. Model Training
6. Model Evaluation

## Models Implemented

- Linear Regression
- Ridge Regression
- Decision Tree Regressor
- Random Forest Regressor

## Results

| Model | RMSE | R² Score |
|---------|---------|---------|
| Linear Regression | 827,105 | 0.86 |
| Ridge Regression | 769,482 | 0.88 |
| Decision Tree | 659,274 | 0.91 |
| Random Forest | 573,974 | 0.94 |

The Random Forest model achieved the strongest performance with an RMSE of approximately 573,974 and an R² score of 0.94.
